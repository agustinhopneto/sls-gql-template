SLS GQL Template is a very simple template to use with AWS Lambda in a very simple way.

<!-- GETTING STARTED -->
## Getting Started
### Running locally
1. Clone the repo
   ```sh
   git clone https://github.com/agustinhopneto/sls-gql-template.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the example
   ```sh
   npm run start
   ```
